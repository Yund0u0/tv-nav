# 📺 看剧导航

给妈妈用的免费看剧网站导航 — 字大、按钮大、点一下就能看。

## ✨ 特点

- 🔤 **大字体** — 标题 28px，站点名 20px，妈妈不用戴老花镜
- 👆 **大按钮** — 触控区 ≥52px，不怕点错
- 📱 **PWA 支持** — 可添加到手机主屏幕，像 App 一样用
- 🎨 **彩色分类** — 国产剧/美剧/韩剧/日剧/动漫/综艺/电影，一目了然
- ⚙ **可自定义** — 左下角齿轮按钮，随时增删网站

## 🚀 部署到 GitHub Pages

```bash
cd Desktop\tv-nav
git init
git add -A
git commit -m "init tv-nav"
git remote add origin https://github.com/Yund0u0/tv-nav.git
git push -u origin master
```

然后在仓库 Settings → Pages → Source 选 master → Save。
